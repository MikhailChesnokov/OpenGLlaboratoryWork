1. Add GLFW
	1.1 Download GLFW from https://github.com/glfw/glfw
	1.2 Copy all the files from "glfw-3.*.*/include/GLFW" to "project_dir/include/GLFW"
	1.3 Build and compile glfw using CMake
	1.4 Place "glfw3.lib" from "glfw-3.*.*/src/Release" to "project_dir/lib"
2. Add gl3w
	2.1 Download gl3w form https://github.com/skaslev/gl3w
	2.2 Run "gl3w_gen.py" (Python 3 needed)
	2.3 Copy all the files from "gl3w/include/GL" to "project_dir/include/GL"
	2.4 Copy "gl3w/src/gl3w.c" to "project_dir"
3. Set up OpenGL version
	3.1 Open application.cpp
	3.2 Find init_app_params method declaration
	3.3 Change major and minor OpenGL version parameters
